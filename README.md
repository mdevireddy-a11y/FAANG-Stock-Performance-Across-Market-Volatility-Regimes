# FAANG Stock Performance Across Market Volatility Regimes

## Overview
This project analyzes how FAANG companies differ in return performance and volatility across market conditions defined by VIX levels.

## Key Questions
- How do stock returns change under stable vs stressed markets?
- Which companies show greater resilience during high volatility?

## Methods
- Calculated daily log returns and volatility using Python (pandas, NumPy)
- Classified market conditions using VIX thresholds
- Compared performance across companies and regimes
- Applied statistical analysis to evaluate differences

## Tools Used
- Python (pandas, NumPy)
- Jupyter Notebook
- Data visualization

## Key Takeaways
- Companies respond differently to market stress
- Some firms show more stable returns under volatility
- Data-driven analysis can reveal resilience patterns in financial markets
